#### 使用 `vercel` 生成静态网站

```
npm i -g vercel # 
vercel init     # Pick an example project
cd <PROJECT>    # Change directory to the new project
vercel          # Deploy to the cloud
```
> 参考 [vercel](https://vercel.com/ "官网")
